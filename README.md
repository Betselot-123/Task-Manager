# Task Manager

A full-stack web application for managing your daily tasks.

## Features
- Add new tasks
- Mark tasks as complete
- Delete tasks
- Tasks persist after page refresh
- Persistent storage
- Responsive design

## Live Demo
Frontend: https://task-manager-nyc8.onrender.com
Backend API: https://task-manager-nyc8.onrender.com/api/tasks
